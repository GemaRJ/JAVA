# Java – Proyectos Académicos y Personales

Este repositorio contiene **ejercicios y proyectos realizados durante mi formación en Desarrollo de Aplicaciones Web (DAW)** y programación en Java. Incluye teoría, prácticas diarias y proyectos personales completos, reflejando la aplicación práctica de los conocimientos adquiridos.

Se destacan aspectos como **manejo de ficheros, orientación a objetos, control de excepciones y lógica de negocio**, utilizando **IntelliJ IDEA** como entorno de desarrollo principal.

---

## 🌟 Tecnologías y herramientas

- **Lenguaje:** Java  
- **Conceptos clave aplicados:**  
  - Orientación a objetos (clases, herencia, polimorfismo)  
  - Manejo de ficheros (lectura y escritura)  
  - Control de excepciones  
  - Lógica de programación y estructuras de datos  
- **Entorno de desarrollo:** IntelliJ IDEA  
- **Control de versiones:** Git y GitHub

## 🚀 Proyectos Personales

Carpeta con **aplicaciones completas**, demostrando la implementación de conceptos avanzados en Java:  
[📂 PROYECTOS PERSONALES](https://github.com/GemaRJ/JAVA/tree/main/Proyecto%20Personales)

### Proyectos destacados

- **Biblioteca:** Sistema de gestión de libros, usuarios y préstamos. Destaca el **uso de ficheros para almacenar datos** y la aplicación de **POO y excepciones**.  
- **Carrera de Coches:** Juego interactivo que aplica **lógica de programación, control de flujo y orientación a objetos**.  
- **Bingo:** Juego de Bingo implementado en Java, reforzando **POO, manejo de colecciones y control de errores**.  

> Estos proyectos muestran **creatividad, planificación y capacidad de implementar programas funcionales y estructurados en Java**, aplicando buenas prácticas de desarrollo.
